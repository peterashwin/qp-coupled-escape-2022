# qp-coupled-escape-2022
Code to produce figures in 2022 manuscript by Ashwin, Creaser and Tsaneva-Atanasova on quasipotentials for coupled escape problems. This was published as Ashwin P, Creaser J, Tsaneva-Atanasova K. (2023) Quasipotentials for coupled escape problems and the gate-height bifurcation, Physical Review E, volume 107, no. 1, article no. 014213, DOI:10.1103/physreve.107.014213
